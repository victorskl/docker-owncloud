# docker-owncloud

### TL;DR

```
git clone https://github.com/victorskl/docker-owncloud.git owncloud
cd owncloud

cp env.sample .env
cp docker-compose.override.yaml.sample docker-compose.override.yaml

docker-compose up -d
docker-compose ps
docker-compose down
```
